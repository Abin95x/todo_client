# Todo Frontend
## Description
This project is a frontend service for managing todos. It uses React.js and Tailwind css for responsive design.

## Technologies Used
- React.js
- For design - Tailwind CSS, Material tailwind, Ant design

  
## Running the App
- To start the application, 
  use:   `npm run dev`
